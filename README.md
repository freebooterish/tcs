# 🐍 贪吃蛇大作战 (Snake Game)

一个具有现代美学设计的网页版贪吃蛇游戏。本项目采用现代 Web 技术构建，具有流畅的动画、精美的视觉效果和完善的游戏逻辑。

## 🌟 功能特点

- **视觉优化**: 
    - 采用卡通蛇头和真实感苹果模型，提升游戏趣味性。
    - 蛇头会根据移动方向动态旋转，视觉效果自然。
    - 霓虹风格 UI 设计，配备毛玻璃效果 (Glassmorphism)。
- **游戏机制**:
    - 支持方向键和 WASD 控制。
    - 具备游戏暂停/继续功能（空格键）。
    - 得分系统与最高分本地存储。
    - 随着得分增加，游戏速度会逐渐加快，增加挑战性。
- **用户系统**:
    - 支持用户名、密码登录以保存记录。
    - 提供游客模式，无需登录即可快速开始。
- **响应式设计**: 
    - 动态背景光球效果。
    - 适配不同尺寸的显示区域。

## 🛠️ 技术栈

- **核心**: HTML5, Canvas API
- **样式**: CSS3 (原生 Vanilla CSS), Google Fonts (Outfit)
- **逻辑**: JavaScript (模块化开发)
- **存储**: LocalStorage (用于保存用户数据和最高分)

## 🚀 快速开始

1. **环境要求**: 任何现代浏览器（Chrome, Edge, Safari, Firefox 等）。
2. **运行**: 直接在浏览器中打开 `index.html` 即可开始游戏。
3. **操作指导**:
    - `上下左右` 或 `WASD`: 控制蛇的移动方向。
    - `空格键`: 暂停或继续游戏。
    - `重新开始` 按钮: 随时重置游戏状态。
